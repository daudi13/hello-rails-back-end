### Getting Started with Rails-API

#### Prequisites 
- browser preferebly chrome
- terminal

run the following commands in your terminal

`git clone https://github.com/daudi13/hello-rails-back-end`

`cd hello-rails-back-end`

`bundle install`

`bin/rails db:create`

`rails s -p 3001`
